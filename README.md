---

## 🔍 Features

- ✅ Scrapes the latest Pastebin pastes
- ✅ Matches against custom keyword list (`keywords.txt`)
- ✅ Sends real-time Telegram alerts using Telegram Bot API
- ✅ Logs all matches with timestamp, keyword, and URL (`logs.csv`)
